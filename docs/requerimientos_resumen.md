# Resumen de requerimientos implementados - ClinCare

## Funcionalidades

- Autenticación de usuarios
- Gestión de colaboradores
- Gestión de pacientes
- Gestión de citas médicas

## Reglas de negocio

- No se permiten usuarios duplicados.
- No se permiten colaboradores duplicados.
- Los médicos deben tener especialidad.
- Los estados válidos son ACTIVO e INACTIVO.
